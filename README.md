# react-native-bob-form

## Built-in Field Types

This package comes standard with a large group of field types that cover all of the common form fields and data types you'll encounter:

#### Text Fields

* **TextType**
* **TextareaType**
* **EmailType**
* IntegerType
* MoneyType
* NumberType
* **PasswordType**
* PercentType
* SearchType
* **UrlType**
* RangeType

#### Choice Fields

* ChoiceType
* EntityType
* CountryType
* LanguageType
* LocaleType
* TimezoneType
* CurrencyType

#### Date and Time Fields

* DateType
* DateTimeType
* TimeType
* BirthdayType

#### Other Fields

* CheckboxType
* FileType
* RadioType

#### Field Groups

* CollectionType
* RepeatedType

#### Hidden Fields

* HiddenType

#### Buttons

* ButtonType
* ResetType
* SubmitType

#### Base Fields

* FormType
